## 编译命令

### 方法1: 使用 XeLaTeX (推荐)

```bash
# 编译一次
xelatex thesis_gold_prediction.tex

# 完整编译（生成目录和引用）
xelatex thesis_gold_prediction.tex
xelatex thesis_gold_prediction.tex
```

### 方法2: 使用 latexmk (自动编译多次)

```bash
latexmk -xelatex thesis_gold_prediction.tex
```

### 方法3: 使用 Overleaf (在线编译)

1. 访问 https://www.overleaf.com
2. 创建新项目 → 上传项目
3. 上传 `thesis_gold_prediction.tex`
4. 设置编译器为 XeLaTeX
5. 点击 "Recompile"

## 系统要求

- **TeX Live** 或 **MiKTeX** 完整版
- **XeLaTeX** 编译器
- **中文字体**: 宋体、黑体、楷体

### macOS 安装

```bash
# 安装 MacTeX (约 4GB)
brew install --cask mactex

# 或者安装基础版
brew install --cask mactex-no-gui
```

### Windows 安装

1. 下载 TeX Live: https://tug.org/texlive/
2. 运行安装程序
3. 选择完整安装

### Linux 安装

```bash
# Ubuntu/Debian
sudo apt-get install texlive-full

# 或安装精简版+中文支持
sudo apt-get install texlive-xetex texlive-lang-chinese
```
